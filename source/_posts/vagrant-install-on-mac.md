簡單的在 Mac 安裝 [Vagrant] 環境

<!--more-->

1. 請先安裝 [Homebrew] - Mac 必裝工具！
2. 用 [Homebrew] 安裝 [Cask] - brew install caskroom/cask/brew-cask
3. 用 [Cask] 安裝 [Vagrant] - sudo brew cask install vagrant
