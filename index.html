2017-08-31

.Net Standard 筆記

簡單筆記

  • 標準的基礎程式庫 API 介面定義
  • 可以跨平台
  • 目前 .Net framework 4.6.1 支援 .Net Standard 2.0 (.Net Standard 1.x 僅有 .Net Code 1.x 支援)
  • .Net Standard 版本控制規則
    • 累加：較新的版本會包含較舊版本的 API
    • 不可變：版本提交後，該版本便已凍結
  • 自行根據 PCL (Portable Class Library)，決定設定檔集合 (就是決定要用哪套實作的程式庫，看是要用 .Net framework 或是 .Net Core)
