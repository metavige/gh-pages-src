Docker service behind proxy - Ubuntu

把以下 export http_proxy 的那一行 unmark, 然後改成你的 proxy 設定

重新啟動 docker service --- service docker restart

(如果你安裝的是 ubuntu 預設的 docker.io，指令就是 service docker.io restart)
